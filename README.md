# BitDrop V9

BitDrop V9 is a lightweight, high‑speed file‑splitting and reconstruction engine designed for reliability, simplicity, and raw performance.  
It takes any file, breaks it into numbered chunks, and reconstructs it perfectly — making it ideal for backups, transfers, testing, and data‑processing workflows.

BitDrop V9 is fully open‑source under the GPLv3 license.

---

## 🚀 Features

- Ultra‑fast file splitting
- Accurate chunk reconstruction
- Simple command‑line interface
- Minimal dependencies
- Works on Windows, Linux, and macOS
- GPLv3 licensed (free and open forever)

---

## 📦 Requirements

- Python 3.10+
- Git (optional, for cloning)
- OS: Windows / Linux / macOS
